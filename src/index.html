зачем изпользуют HTML-вёрстку?

HTML-верстка сайтов нужна для создания полноценных веб-страниц\сайтов, для последующего размещения оных в сети интернет на всеобщее обозрение. Как правило верстальщик делает свою верстку на основе дизайнерского макета сайта предоставленного клиентом, исходя из его потребностей.

LinFolio.ru
Сайт-портфолио HTML-верстальщика, где вы можете заказать качественную и адаптивную верстку для вашего сайта по PSD макетам. Принимаются заказы на вёрстку Лендингов, цельных сайтов, правки по сайту, изменение дизайна сайта, устранение багов верстки, и верстку отдельных страниц без посадки на CMS.
